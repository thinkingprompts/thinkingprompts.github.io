Root Cause Analysis

6Ms

  • Mother Nature (Environment)
  • Man/Mind Power (People, physical or knowledge work)
  • Machine (equipment, technology)
  • Materials (raw material, consumables, and information)
  • Methods (processes)
  • Measurement (inspection)

5Whys

  • Ask why 5 times

Process

  • Material Flow: connection, direction, temperature, pressure, flow rate, composition
  • Energy Flow: electricity, heat
  • Data Flow: hardware, software
  • Control Flow: hardware, software

Change Analysis

  • Analyze the changes leading up to an event
  • Compare a process that has failed to one that is operating correctly
  • Compare conditions when a process failed to when it is operating correctly

Locating Errors

  • Drawing of physical connection
  • Creation of sequence of events
  • Use of analysis points: retrieve information from different locations
  • Backtracking: trace error backward from where it happened in the sequence of events
  • Divide and conquer: use binary search locate error
  • Simplication: eliminate portions that are not relavent to error
  • Clustering of errors: group errors into different classes
  • Forward analysis: trace error forward to where it happened
